## Nodejs Task App

#### Installation
  ```javascript
 npm install
```

#### Usage
##### Run commands in terminal
1. Add Task
 ```javascript
 node app.js add --title="TITLE NAME" --body="BODY"
```
2. Delete Task
 ```javascript
 node app.js delete --title="TITLE NAME"
```
3. Search Task
 ```javascript
 node app.js search --title="TITLE NAME"
```
4. List all Tasks
 ```javascript
 node app.js list 
```
